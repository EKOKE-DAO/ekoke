# Ekoke ERC20

## Verify source code

put Etherscan apikey in `.env` file.

Then run:

```sh
yarn hardhat verify --network goerli "$CONTRACT_ADDRESS" "$OWNER_ADDRESS" $SWAP_FEE
```
